#Projects

Personal projects
I have recently become interested in programming with a Raspberry Pi and Arduinos and have enjoyed working on a few personal projects such as Halloween decorations, an autonomous vehicle, and an IOT device.

Flashing Jack-O-Lantern
I took an existing jack-o-lantern decoration I had and added flashing lights and audio, which can be triggered from a webapp. This project uses a Raspberry Pi, Blinkt LED strip, and a small speaker. The webapp is hosted on the Raspberry Pi using Flask for the webserver. There are several preprogrammed options for the different sounds and lights. The light patterns are simply changes between three colors, with random delays between flashes and random brightness for each flash. Everything is written using Python.

Halloween Strobe Light
This was my first project after getting a Raspberry Pi. This uses just the Raspberry Pi and a Blinkt LED strip. Once the script is started, it simply randomly selects a color and a brightness to flash, as well as a random delay between flashes to create a 'Rainbow Strobe Light'. The script would auto run when the Pi started up, and I was able to power the Pi through a portable battery pack which made it portable and perfect for a front window backdrop.
